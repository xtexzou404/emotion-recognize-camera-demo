# 🎓 Emotion Recognition System for ICB Students

Система распознавания эмоций студентов колледжа ICB с использованием веб-камер видеонаблюдения. Проект позволяет анализировать эмоции студентов в реальном времени через веб-интерфейс.

## 📌 Описание проекта

Данный проект представляет собой веб-приложение для распознавания эмоций студентов с камер видеонаблюдения. Система анализирует выражение лица и определяет эмоциональное состояние пользователя.

Приложение состоит из:

* **Frontend** – интерфейс пользователя
* **Backend** – серверная часть
* **Модуль распознавания эмоций** – анализ лица

## ⚙️ Установка

### 1. Клонировать проект

```bash
git clone https://github.com/xtexzou404/emotion-recognize-camera-demo.git
cd emotion-recognize
```

### 2. Установить зависимости

#### Backend

```bash
cd backend
npm i
```

#### Frontend

```bash
cd frontend
npm i
```

## ▶️ Запуск проекта

### Запуск Backend

```bash
cd backend
npm start dev
```

### Запуск Frontend

```bash
cd frontend
npm start dev
```
